Detailed Description

A sketch for reading sensor values and sending it with Bluetooth to a serial monitor.

Grijpersensor.ino is a sketch to connect the HC-06 module and communicate using the serial monitor. When the HC-06 is powered, the HC-06 defaults to communication mode. The default baud rate for communication mode is 9600.

The bluetooth SoftwareSerial for the bluetooth HC-06.

The first argument is RX on the Arduino. The second argument is TX on the Arduino. The RX and TX of the Arduino are connected to the TX and RX of the Bluetooth, respectively. The Arduino-TX is connected to the Bluetooth-RX through a voltage divider
